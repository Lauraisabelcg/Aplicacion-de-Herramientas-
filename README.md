# Aplicacion Registro de Herramientas 🔨

Aplicacion creada para que los residentes de un conjunto puedan:
 * Solicitar el uso de herramientas.
 * Enviar PQRS al administrador 
 
 PD:Solo los usuarios agregados previamente por el administrador pueden hacer uso de dichas herramientas.Se guarda registro de herramientas, usuarios, mensajes y solicitudes realizadas por los usuarios.

 ***

 ## Funcionalidades 📝

**Rol administrador**  
* Registrar y/o Eliminar usuarios para que puedan usar las herramientas.
* Agregar y ver la lista de Herramientas.
* Leer PQRS realizada por los usuarios.
* Consultar el historal de prestamos de herramientas.

**Rol Usuario**
* Ver lista de Herramientas Actuales.
* Solicitar herramientas.
* Escribir PQRS al administrador.
* Consultar el historal de prestamos de herramientas.
